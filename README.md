# `1. Hello Docker`

## `1.0. Images`

### 1.0.1. Add

```bash
  ▶ docker pull [image_name]:[version]
```

### 1.0.2. Remove

```bash
  ▶ docker image rm [image_name]:[version]
```

### 1.0.3. All

```bash
  ▶ docker image
```

## `1.1. Containers`

### 1.1.0. Add

```bash
  ▶ docker create [image_name]
```

### 1.1.0. Execute

```bash
  ▶ docker start [image_id]
```

### 1.1.0. Show

```bash
  ▶ docker ps !(-a)
```

<b>`stop container`</b>

```bash
  ▶ docker stop
```
